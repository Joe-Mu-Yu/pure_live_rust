## 🔧 技术挑战与解决方案

### 1. MPV 竖屏渲染

**挑战**: MPV 默认不支持自动竖屏切换

**解决方案**:
```rust
// 检测视频方向
let orientation = check_orientation();

// 自动设置旋转
if orientation == Portrait {
    mpv_set_property_string("video-rotate", "90");
}
```

### 2. Protobuf 弹幕解析

**挑战**: B 站/抖音使用 Protobuf 协议

**解决方案**:
- 使用 `prost` 库重新编译 `.proto` 文件
- 参考原项目 Dart 实现

### 3. 跨平台打包

**挑战**: Android/macOS 打包配置复杂

**解决方案**:
- Tauri 已内置跨平台支持
- 配置 `tauri.conf.json` 即可
